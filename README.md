🖼️ CollageMaster — создатель коллажей с автоматической компоновкой
Мощный инструмент для создания коллажей из нескольких изображений с автоматической компоновкой, настраиваемыми сетками, отступами, рамками, тенями и подписями.
Реализован на 7 языках программирования для демонстрации алгоритмов компоновки и работы с графикой.

🧠 Концепция
CollageMaster — это интеллектуальный генератор коллажей, который автоматически:

✅ Анализирует размеры входных изображений и определяет оптимальную сетку.

✅ Компонует изображения в сетке с заданным количеством столбцов или автоматическим расчётом.

✅ Масштабирует каждое изображение, сохраняя пропорции, с возможностью обрезки под размер ячейки.

✅ Добавляет отступы между изображениями и рамки вокруг каждого.

✅ Применяет тени для объёмного эффекта (опционально).

✅ Добавляет подписи к изображениям (на основе имён файлов или пользовательского текста).

✅ Экспортирует результат в PNG или JPEG с высоким качеством.

🚀 Как запустить
Каждая версия использует соответствующую библиотеку для работы с изображениями. Инструкции по установке и запуску приведены ниже.

bash
# Python (Pillow)
pip install Pillow
python collage_python.py

# C++ (OpenCV)
# Требуется установка OpenCV и компиляция:
g++ -std=c++17 collage_cpp.cpp -o collage `pkg-config --cflags --libs opencv4`
./collage

# Java (стандартный AWT)
javac collage_java.java && java collage_java

# C# (System.Drawing)
dotnet add package System.Drawing.Common
dotnet run

# Go (image/png, image/jpeg)
go run collage_go.go

# Rust (image crate)
cargo build --release && ./target/release/collage_rs

# JavaScript (node-canvas)
npm install canvas
node collage_js.js
🧩 Пример использования
bash
# Создать коллаж из всех изображений в папке images с 3 колонками, отступами 10px, рамкой 2px, тенями и подписями
collage --input images --output collage.png --cols 3 --padding 10 --border 2 --shadow --labels

# Автоматически определить количество колонок
collage --input images --output collage.png --auto-cols
Результат: collage.png — красивый коллаж со всеми изображениями.
